## 🔍 Understanding Subnet Data

The tool translates complex network records into readable text. You see the following values:

- UIDs: The unique identifier for a node in a subnet.
- Weights: The current influence of a node in the network.
- Version: The software version running on the network peer.
- Metadata: Key information about the subnet configuration.
